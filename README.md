# ⚡ ADVANCED PORT SCANNER
> **Ultimate Multi-Threaded Security Suite with Interactive Live Radar Visualizer**

An enterprise-grade, high-performance network diagnostic and security auditing tool built entirely in Java Swing. Featuring a modern cyberpunk **Deep Slate & Electric Green** glossy user interface, this suite integrates advanced asynchronous port scanning, multithreaded architecture, real-time visual tracking, and automated security vulnerability mappings.

---

## 🚀 Key Features

* **⚡ Ultra-Fast Multithreading:** Powered by a dynamic Java `ExecutorService` thread pool allowing parallel socket auditing (up to 150+ threads) with zero UI freezing.
* **📡 Interactive Live Radar Matrix:** A custom `Graphics2D` animated radar sweep sub-panel that visually renders active host entry nodes as glowing neon indicators in real-time.
* **🌐 Active Host Discovery Engine:** Performs pre-scan structural ping validation via ICMP/TCP fallback vectors to confirm target node availability.
* **🧠 Heuristic OS Fingerprinting:** Automatically deduces the architecture of the target host (Windows NT vs. Linux Kernel vs. Generic/Embedded) based on active port signatures.
* **🔒 Integrated CVE Exploit Research:** Embeds dynamic, interactive triggers within the results table that link discovered active ports directly to the **National Vulnerability Database (NVD)** for quick exploit identification.
* **💾 Enterprise Log Exporter:** Safely compiles and outputs detailed structured audit reports into global `.txt` formats.
